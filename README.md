# 📚 Origamid — HTML e CSS para Iniciantes

Repositório dedicado ao armazenamento de exercícios, projetos práticos e anotações desenvolvidos durante o curso **HTML e CSS para Iniciantes** da Origamid.

Este repositório faz parte da minha formação em Desenvolvimento Front-End, com foco na construção de uma base sólida em estruturação e estilização de páginas web.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsividade
- Organização e boas práticas de código

---

## 🎯 Objetivos do Curso

- Compreender a estrutura semântica do HTML
- Aplicar estilos com CSS de forma organizada
- Criar layouts modernos e responsivos
- Desenvolver base sólida para avançar para JavaScript e React

---

## 📈 Evolução

Este repositório representa minha evolução no desenvolvimento Front-End, consolidando fundamentos essenciais e preparando o caminho para tecnologias mais avançadas.
