# 🎬 Projeto Cinema

Projeto desenvolvido como exercício prático do curso da Origamid, com foco em estruturação em HTML e estilização com CSS.

---

## 📚 Sobre o Projeto

O projeto **Cinema** simula um site simples com:

- Página inicial
- Páginas para dois estados (Rio de Janeiro e São Paulo)
- Navegação entre páginas
- Estilização externa com CSS

O objetivo é praticar:

- Estrutura básica de HTML
- Organização de pastas
- Links relativos
- Uso de classes no CSS
- Separação entre estrutura (HTML) e estilo (CSS)

---

## 📁 Estrutura do Projeto

```
cinema/
│
├── index.html
│
├── estados/
│   ├── rj.html
│   └── sp.html
│
└── css/
    └── style.css
```

---

## 🖥️ Funcionalidades

### Página Inicial

- Título da aba: **Cinema**
- `<h1>` com o nome do projeto
- Links para cada estado dentro de `<h2>`

### Páginas dos Estados

Cada página contém:

- `<h1>` com o nome do estado
- Lista de filmes com horário e dia
- Indicação visual diferente para filmes de hoje e amanhã
- Link para voltar ao início

---

## 🎨 Estilização

O arquivo `style.css` aplica:

- Fonte Arial em todo o site
- Parágrafos com 20px
- Filmes de **Hoje** em `darkred`
- Filmes de **Amanhã** em `darkblue`
- Links sem sublinhado
- Links dentro de `<h2>` com cor `seagreen`

---

## 🎯 Objetivo do Exercício

- Praticar HTML básico
- Trabalhar com múltiplas páginas
- Utilizar CSS externo
- Aplicar boas práticas de organização de projeto
