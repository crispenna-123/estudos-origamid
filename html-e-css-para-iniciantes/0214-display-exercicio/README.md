# 🚗 Projeto Carros e Bicicletas

Exercício prático do curso da Origamid para treinar HTML e CSS.

---

## 📚 Sobre

Página simples com dois produtos:

- Carro
- Bicicleta

Cada produto possui:

- Nome
- Preço
- Descrição
- Botão de compra

---

## 📁 Estrutura

carros-bicicletas/
│
├── index.html
└── style.css

---

## 🎨 Estilo

- Fonte Arial
- Texto centralizado
- Preço e botão na cor `#febe01`
- Texto do botão `#332200`
- Texto padrão `#111`
- Bordas arredondadas (`4px`)
- Borda dos cards `#b2b2b2`

---

## 🎯 Objetivo

- Praticar HTML e CSS básico
- Trabalhar com cores personalizadas
- Aplicar classes no CSS
- Criar estrutura organizada e reutilizável

---

Projeto para fins educacionais.
